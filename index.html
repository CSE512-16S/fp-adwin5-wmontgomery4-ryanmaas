Interactive Learning for Hierarchies of Concepts

We built a visualization system for a crowdsourcing algorithm which learns hierarchies of concepts.

  • Building Hierarchies of Concepts via Crowdsourcing

The system learns by obtaining answers to simple questions such as "Is orange a type of citrus fruit?".

Hierarchies are represented by trees, where a node is considered a subclass of all its ancestors.

Given the responses, a probabilistic model defines a distribution over possible hierarchies, but the algorithm only returns the maximum a posteriori (MAP) tree, which may not represent the entire distribution due to uncertainty and sampling error.
